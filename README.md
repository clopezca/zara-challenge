# Mobile phone catalog

A web application for browsing, searching and managing a mobile phone catalog, built as part of a technical challenge.

## Tech stack

- **React 18** with TypeScript
- **Vite** — dev and production build tool

## Getting Started

### Prerequisites

- Node.js >= 18
- npm >= 9

### Installation

```bash
git clone
cd zara-challenge
npm install
```

### Development mode

```bash
npm run dev
```

### Production mode

```bash
npm run build
npm run preview
```
